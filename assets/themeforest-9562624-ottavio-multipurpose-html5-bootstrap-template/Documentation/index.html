Tips

Due to local security restrictions, some things will not work locally, especially in Chrome. Everything will work once the files have been uploaded to your webserver.

Firefox's Firebug extension or Chrome's console are highly recommended for debugging purposes.

Do not start from scratch! Clone an existing page from the demo and modify it to learn how it works.

Are you getting an error message? Someone else might have seen it too, try a Google search for a quick fix.

Explore the live demo for pages ideas and sample code.

HTML Structure

Grid system

Being based on Twitter Bootstrap, Ottavio inherits its awesome grid system. It's a responsive, mobile first fluid grid that appropriately scales up to 12 columns as the device or viewport size increases. Example markup:

<div class="row">
    <div class="col-md-12">
        Level 1 column
        <div class="row">
            <div class="col-md-6">Level 2</div>
            <div class="col-md-6">Level 2</div>
        </div>
    </div>
</div>

Please take a look at Bootstrap's documentation: http://getbootstrap.com/css/#grid

Spacing between block elements

To easily separate block elements, Ottavio relies on internal padding. The sep-top and sep-bottom classes manages the top and bottom padding respectively, for maximum flexibility. The actual amount of space is specified by the -xs, -sm, -md, -lg, -2x, -3x, -4x and -5x modifiers.
Example code:

<div class="sep-top-lg sep-bottom-lg">
  <!-- this element has medium spacing above and below -->
</div>
<div class="sep-top-sm sep-bottom-4x">
  <!-- this other element has small spacing above and a huge spacing below -->
</div>

Customization & Styles

Colors

Several color variations are included with the theme. You can use them by linking the relevant stylesheet after the main stylesheet in the document's <head> section:

<!DOCTYPE html>
<html class="no-js">
  <head>
    ...
    <!-- this is Ottavio's main stylesheet: -->
    <link rel="stylesheet" href="styles/main.css">
    <!-- link here the color variation stylesheet: -->
    <link rel="stylesheet" href="styles/theme_meadow.css">
  </head>
  <body>
  ...
  </body>
</html>

These are the predefined variations:

  • theme_meadow.css
  • theme_royalblue.css
  • theme_tangerine.css
  • theme_sangria.css
  • theme_eggplant.css
  • theme_lime.css

You can create your own variations by cloning one of these stylesheets, and substituting the color codes.

Layout Style

The template has two different layout modes, Full-screen and Boxed. Full-screen is the default option, if you'd like to use the Boxed version, add the following class to the <body>:

<body class="boxed-layout">

Custom styles

If you need to define you own classes, we strongly recommend to put them in a separate file. This way it'll be safe to update the theme without breaking your work. Make sure you include it after the main stylesheets, so it gets higher priority. Example code:

<!DOCTYPE html>
<html class="no-js">
  <head>
    ...
    <link rel="stylesheet" href="styles/main.css">
    <link rel="stylesheet" href="styles/theme_meadow.css">
    <!-- add here your own stylesheet: -->
    <link rel="stylesheet" href="my_custom_rules.css">
  </head>
  <body>
  ...
  </body>
</html>

Icons

Font Awesome is a @font-face iconset that allows you to change size and color of the icons just using CSS.

To add an icon, simply create an empty element with class 'fa fa-[icon name]', for example:

<i class="fa fa-file"></i>
<i class="fa fa-home"></i>
<i class="fa fa-envelope"></i>

A complete list of all available icons can be found here: http://fontawesome.io/icons.

Sliders

Home Page Sliders

Slit Slider

Slit Slider is a slideshow with a twist: when navigating the slides, the current one will be "cut open" into two slices and moved away, revealing the next or previous slide. The slider can be used in a responsive context and it has been updated and improved.

Further informations, along with a complete tutorial, can be found at http://tympanus.net/codrops/2012/10/24/slit-slider-revised/

Content Sliders

OWL Carousel is a touch enabled jQuery plugin that lets you create beautiful responsive carousel slider. Implementation is very simple: add the "owl-carousel" class to your container, then wrap each slide in an element with class "item", and you're good to go.

Options can be set as 'data-' attributes on the container. Some options are:

data-single-item="[true/false]"
Display only one item per slide.
data-navigation="[true/false]"
Display "next" and "prev" buttons.
data-auto-play="[true/false]"
Wheter the carousel should rotate automatically. You can set it as an amount of milliseconds, or set it as true (which defaults to 5 seconds).

More options are available, please refer to the documentation.

Example Markup:

<div class="owl-carousel" data-navigation="false" data-single-item="true" data-auto-height="true" data-auto-play="false">
  <!-- Start Item Slide-->
  <div class="item">
    First Item
  </div>
  <!-- End Item Slide-->
  <!-- Start Item Slide-->
  <div class="item">
    Second Item
  </div>
  <!-- End Item Slide-->
  <!-- Start Item Slide-->
  <div class="item">
    Third Item
  </div>
  <!-- End Item Slide-->
</div>

Lightbox

The lightbox functionality is driven by PrettyPhoto. All you have to do is link the bigger image to in the anchor's href, then add a data-rel attribute with a value of "prettyPhoto". Example Markup:

<a href="bigimage.jpg" data-rel="prettyPhoto">...</a>

If you need the gallery functionality (with next/previous buttons) you can append the gallery name in square brackets after prettyPhoto, like so:

<a href="bigimage.jpg" data-rel="prettyPhoto[mygallery]">...</a>

Please check prettyPhoto's documentation.

PHP Scripts

Regarding contact form as main part of Contact Us section, only thing you need to do is to setup your email address.
To do this, please open "php/contact-form.php" file and change this line:

$to = 'you@domain.com';

Please note: this will be functional only on your server, php files can not be executed on your local computer!

Portfolio wall

Portfolio Images

For best results, the thumbnails must be in square (1:1) format. You can spice up the layout by adding items that have either double height (1:2), double width (2:1) or both; those special items should have a class of .height2x and/or .width2x. Obviously, in those cases the image should be in 1:2 of 2:1 format as well.

Example markup:

<ul id="isotope" class="portfolio isotope">
  <li class="item">
  <!-- this is a regular square item -->
  </li>
  <li class="item height2x">
  <!-- this item is twice as tall as the others -->
  </li>
  <li class="item width2x">
  <!-- this item is twice as large as the others -->
  </li>
  <li class="item height2x width2x">
  <!-- this item is twice as big as the others. Use sparingly! -->
  </li>
</ul>

Spaced grid

If youd'like to put some space between the items, wrap the item's content in an .item-wrapper.

Please note: in this case, items of different size may break the grid. Stick with equal-sized items if you want the spaced grid, or switch to the non-spaced version if you like items of different size (see above).

Example markup:

<ul id="isotope" class="portfolio isotope">
  <li class="item">
    <div class="item-wrapper">
    ...
    </div>
  </li>
</ul>

Hover effects

By default, when hovering an item a dark mask will cover the image. You can choose the white variant by adding a .mask-light to the mask element. Example markup:

<ul id="isotope" class="portfolio isotope">
  <li class="item">
    <img class="img-responsive" src="...">
    <div class="mask">
      <div class="mask-content">
        <!-- this item has the regular mask -->
        ...
      </div>
    </div>
  </li>
  <li class="item">
    <img class="img-responsive" src="...">
    <div class="mask mask-light">
      <div class="mask-content">
        <!-- this item has the white mask -->
        ...
      </div>
    </div>
  </li>
</ul>

Filtering the Portfolio

Filtering of the Portfolio items is handled by Isotope. Add a descriptive class to the items, then create a filter button bound to the class you want to filter via a data-filter attribute. Example markup:

<div id="filters" class="light text-center sep-top-lg">
  <button data-filter="*" class="btn btn-xs upper btn-primary">show all</button>
  <button data-filter=".design" class="btn btn-xs upper">design</button>
  <button data-filter=".photography" class="btn btn-xs upper">photography</button>
  <button data-filter=".videography" class="btn btn-xs upper">videography</button>
</div>

The button with the .btn-primary class is the one that sets the default filtering. In this example it's bound to the "show all" button, but you may want another filter selected by default.

The items: 
<ul id="isotope" class="portfolio isotope">
  <li class="item videography">...</li>
  <li class="item photography">...</li>
  <li class="item photography design">...</li>
</ul>

Pleas refer to Isotope's documentation here: http://isotope.metafizzy.co.

Ajax Portfolio Expander

Each portfolio item contains a link to a page with its details. This is handled via an Ajax request, loading the content just above the Portfolio. Within this area, navigation to the next or previous project is automatically handled via javascript.

Please note: The portfolio expander is made with ajax and loads an external HTML. This server request can't be performed locally. Once uploaded to your virtual host it will work as expected.

Also note that Ottavio assumes it's installed in the root folder of your server, and the portfolio files are located in /portfolio. If this is not your case, you should manually configure the portfolio path. You do this by:

  1. setting the correct data-folder attribute on the container, reflecting the Portfolio location relative to your root folder.
  2. setting the absolute (full) path of the details page (relative to your root folder) for each item, preceded by #!.

Example markup:

<!-- setting the portfolio folder location on the container section -->
<section data-folder="/my/portfolio/folder" id="portfolio-gallery">
  <!-- start ajax section -->
  ...
  <!-- end ajax section -->
  <div class="article-body-content">
    <ul class="portfolio isotope" id="isotope">
      <!-- start item -->
      <li class="item">
        <img class="img-responsive" alt="" src="img/my_image.jpg">
        <div class="mask">
          <div class="mask-content">
            <div class="mask-wrapper text-center">
              <!-- link to the details page, preceded by #! -->
              <a class="info ajax_load" href="#!/my/portfolio/folder/my_project_page.html">
                <i class="fa fa-link fa-border"></i>
              </a>
              ...
            </div>
          </div>
        </div>
      </li>
      <!-- end item -->
    </ul>
  </div>
</section>

Deep linking is supported, so appending #!/portfolio/my_project.html to your page's URL will automatically expand that project. The brower's "back" button will also work.

To properly work, the linked .html should have a structure similar to this:

<div id="ajaxpage" class="ajaxpage sep-top-2x sep-bottom-2x">
  <div class="row">
    ...Project photos and specs...
  </div>
</div>

Mind the .ajaxpage class, it's needed for the script to fetch the content. This block can be wrapped inside other html content, what's important is just the .ajaxpage div.

Should you have any problem, Firefox's Firebug extension or Chrome's console are of great help in figuring out what's going on: they will show you what the browser is trying to reach, and what it gets back. Please check them out before requiring support.

Google Maps

We've included a jQuery plugin to easily set up a Google Map. All you have to do is configure the options with a Javascript object, the plugin will take care of the rest.

Just make sure the container has a map-canvas id. Example markup:

<div id="map-canvas" style="height:500px">
  <script type="text/javascript">
    var
      lat = -37.817921,  // your latitude here
      lon = 144.965076,  // your longitude here
      infoText = [
        '<div style="white-space:nowrap">',
          '<h5>OTTAVIO STUDIOS</h5>',                // put here the markup
          'Envato, level 13, 2<br>',                 // you want to appear
          'Elizabeth St, Melbourne, Victoria<br>',   // inside the map's
          '3000, Australia',                         // infowindow
        '</div>'
      ],
      mapOptions = {
        scrollwheel: false,
        markers: [
          { latitude: lat, longitude: lon, html: infoText.join('') }
        ],
        icon: {
          image: "img/marker.png",                  // you can change the marker icon
          iconsize: [72, 65],
          iconanchor: [12, 65],
        },
        latitude: lat,
        longitude: lon,
        zoom: 11
      };
  </script>
</div>

The plugin's documentation can be found here: http://labs.mario.ec/jquery-gmap/.

Heads up! The height of the map should be declared as style attribute.

Device Mockups

You can quickly create a virtual mobile device to be used as a fancy frame for your images. The type of device, its orientation and color are set via data-device, data-orientation, and data-color attributes, respectively. Example markup:

<div class="device-mockup" data-device="iphone5" data-orientation="portrait" data-color="black">
    <div class="device">
        <div class="screen">
            <!-- PUT CONTENTS HERE -->
        </div>
        <div class="button">
            <!-- You can hook the "home button" to some JavaScript events or just remove it -->
        </div>
    </div>
</div>

More examples can be found here.

Effects

Parallax scrolling

To create a parallax background, simply add a the parallax class to the element. The background image should be declared as style attribute.

<section style="background-image: url('img/myimage.jpg');" class="header-section parallax">
  <div class="section-shade">
    <div class="container">
      ...
    </div>
  </div>
</section>

The .section-shade element adds a black gradient above the background that makes the text easier to read.

For best results, the background image should have a dimension of at least 1500x750 pixel.

Warning: parallax scrolling isn't supported by mobile devices.

Fading title

To get a nice fading effect when scrolling page titles, add the fading-title class to the header. Example:

<section class="header-section fading-title">
  <div class="section-shade sep-top-4x sep-bottom-4x">
    <div class="container">
      <div class="section-title upper text-center light">
        <h2>My Awesome Title</h2>
        <p class="lead">Nullam quis risus eget urna mollis ornare vel eu leo.</p>
      </div>
    </div>
  </div>
</section>

Entrance animations

You can add some animation to the elements which fires when they scroll into view. Just add two classes: wow, and the effect you want to apply. Optionally, you can specify how much the animation should be delayed via the data-wow-delay attribute. Example markup:

<div class="wow bounceInLeft" data-wow-delay=".5s">
    ...
</div>

Please refer to animate.css for a complete list of available effects.

WOW examples and documentation: http://mynameismatthieu.com/WOW/
